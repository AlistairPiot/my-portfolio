## Getting stated 🚀

### Prerequisites 🔗

-   [Git](https://git-scm.com)
-   [Node.js](https://nodejs.org/en/download/) (which comes with [NPM](http://npmjs.com))
-   [Yarn](https://yarnpkg.com/) instead of NPM

## Setup 🔧

### Create the project 📁

```bash
yarn create vite
```

### Install dependancies 🔖

```bash
yarn
```

### Start the development server 💻

```bash
yarn run dev
```
